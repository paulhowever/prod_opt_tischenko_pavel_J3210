# Production Optimization --- Lab 1

Реализация типа данных **конструктивное число** и исследование методов
оптимизации для вычислимых функций.

Проект включает:

-   реализацию арифметики конструктивных чисел\
-   набор тестовых функций (черные ящики)\
-   методы оптимизации\
-   экспериментальное исследование сходимости

------------------------------------------------------------------------

## Конструктивное число

Конструктивное число представляется интервалом

    (a, b),  a, b ∈ Q

где истинное значение числа лежит внутри интервала.

Вещественное приближение вычисляется через параметр

    α ∈ [0,1]
    x = a + α(b - a)

Тип поддерживает нативные операции:

    +
    -
    *
    /

а также сравнение и получение вещественного приближения.

------------------------------------------------------------------------

## Тестовые функции

В проекте реализованы функции для исследования методов оптимизации:

-   квадратичная функция с хорошим числом обусловленности\
-   квадратичная функция с плохим числом обусловленности\
-   функция Розенброка

Все функции имеют **унифицированный интерфейс вызова**, что позволяет
использовать их с любыми реализованными методами оптимизации.

------------------------------------------------------------------------

## Методы оптимизации

В проекте реализованы:

-   **Gradient Descent**
-   **Nelder--Mead**
-   **Newton Method**

Методы применяются для поиска минимума тестовых функций.

------------------------------------------------------------------------

## Исследование

В ходе экспериментов проводится сравнение методов по следующим метрикам:

-   скорость сходимости\
-   количество итераций\
-   количество вызовов функции\
-   точность найденного минимума

Дополнительно анализируется рост погрешности `ε` при работе с
конструктивными числами.
